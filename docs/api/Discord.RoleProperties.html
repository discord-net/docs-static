Examples

The following example uses ModifyAsync to apply changes specified in the properties,

var role = guild.GetRole(id);
if (role == null) return;

await role.ModifyAsync(x =>
{
    x.Name = "cool boi";
    x.Color = Color.Gold;
    x.Hoist = true;
    x.Mentionable = true;
});
